# AI Dashboard

## Setup:
```bash
npm install
npm run dev
```

## Deploy to Vercel:
1. Push this repo to GitHub
2. Go to https://vercel.com
3. Click "New Project" and import this repo
4. Deploy

## Features
- Assistant tab (GPT-4)
- Dashboard with job/task stats
- Uploads tab with reels/captions links
- Withdraw button
